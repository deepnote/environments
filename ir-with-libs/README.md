# R image with preinstalled libraries
This image is based on `deepnote/ir` and installs the following libraries:
- tidyverse 
- data.table 
- RSQLite 
- remotes 
- reticulate 
- igraph 
- plotly 
- rgl 
- hdf5r 
- rJava

See the [Dockerfile](https://github.com/deepnote/environments/blob/main/ir-with-libs/Dockerfile).
